# customer-management

### A Full-Stack Application using Thymeleaf , Spring Boot and PostgreSql.

I used;

* **Thymeleaf** as fronted
* **Spring Boot**  as backend
* **PostgreSQL** as database
* **Lombok** to get rid of code clutter
* **Bootstrap**  as UI
